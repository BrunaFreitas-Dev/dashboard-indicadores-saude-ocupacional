# 📊 Dashboard de Indicadores de Saúde Ocupacional

Projeto desenvolvido com Power BI com o objetivo de analisar dados simulados de colaboradores e gerar indicadores estratégicos relacionados a afastamentos.

---

## 🎯 Objetivo do Projeto

Analisar dados de colaboradores e identificar padrões relacionados a afastamentos, permitindo visualização estratégica para apoio na tomada de decisão.

---

## 📌 Indicadores Desenvolvidos

- Total de Funcionários
- Total de Afastamentos
- Média de Dias Afastados
- Taxa de Afastamento (%)

---

## 📊 Análises Realizadas

- Distribuição de afastamentos por departamento
- Análise dos principais motivos de afastamento
- Comparação entre setores
- Filtros interativos por departamento, cargo e motivo

---

## 🛠 Tecnologias Utilizadas

- Power BI
- Excel
- DAX
- Modelagem de Dados
